Important FAQ

What is the document object model

DOM represent the page content as HTML, allowing scripts to manipulate its content and structure.

How do you select an element from the DOM

lots of way, such as:

  1. getElementById
  2. getElementsByClassName
  3. getElementsByTagName
  4. querySelector
  5. querySelectorAll
What is event delegation in the context of the DOM, and why it is useful

It's a technique, where a single event listener is added to a parent element to handle events on its child elements.

How do you manipulate an element's attributes and styles using the DOM

for manipulatean element's attributes using, setAttribute/ getAttribute and for style we can use element.style.property
